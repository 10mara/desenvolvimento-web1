Tags de código HTML

Tag Code

A code tag é usada para definir um trecho de código de computador,ou seja, exibir uma parte do código de programação. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Exemplo:

            
            #Crie um programa que simule uma calculadora básica.
            #O usuário deve informar dois números e a operação desejada (+, -, *, /).
            #O programa deve realizar o cálculo e exibir o resultado.
            #---------------------------------------------#
            
            print ("Calculadora Básica \n")
            num1 = eval(input("Digite o primeiro número:  \n"))
            num2 = eval(input("Digite o segundo número:  \n"))
            operacao= input("Digite qual operação vc deseja(soma,sub, mult,div): \n ")
            
            soma = num1+num2
            sub = num1-num2
            mult = num1*num2
            div = num1/num2
            
            if operacao=='soma':
                print(" O resultado desta soma é:",soma)
            elif operacao=='sub':
                print(" O resultado desta subtração é:",sub)
            elif operacao=='mult':
                print(" O resultado desta multiplicação é:",mult)
            elif operacao=='div':
                print(" O resultado desta divisão é:",div)

Tag Samp

A samp tag é usada para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Exemplo:

            Calculadora Básica 

                Digite o primeiro número:  
                120
                Digite o segundo número:  
                2
                Digite qual operação vc deseja(soma,sub, mult,div): 
                 div
                 O resultado desta divisão é: 60.0

Tag Pre

A pre tag define texto pré-formatado. O texto em um pre elemento é exibido em uma fonte de largura fixa e preserva espaços e quebras de linha. O texto será exibido exatamente como está escrito no código-fonte HTML.

Exemplo:

            
            Ctrl+ - Cria um arquivo novo.
            Ctrl+O - Abre um arquivo existente.
            Ctrl+S - Salva as alterações.
            Ctrl+Shift+S - Salva no dispositivo.
            Ctrl+C - Copia.
            Ctrl+V - Cola o item na área de tranferência.
            Home - vai ao início do texto/linha.
